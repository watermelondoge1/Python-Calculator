# Python Calculator

A calculator application built in Python.

## Features 
Currently:

```markdown
- Basic openable app with UI
- Numbers 0-9
- 4 main operators (+, -, *, /)
- Basic error handling for divison by zero and invalid syntax entered
- Button grid layout
```

## Planned Features

```markdown
- Improved UI
```

## How To Run

- Clone this repository: 
```bash 
git clone https://github.com/watermelondoge1/Python-Calculator.git
```
- Install required packages: pip install -r requirements.txt
- Run application: python UI.py
